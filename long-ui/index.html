# long-ui

# 一个基于Vue2.x的移动端UI long-ui

Demo: http://www.weberlong.com/

# 技术栈

vue2 + vue-router + webpack + icomoon + less + jsx

  • MVVM框架：Vue 2.0
  • 脚手架：vue-cli
  • 构建工具：webpack
  • 前端路由：vue-router
  • 图标icons: icomoon
  • 代码风格检查：eslint
  • 源码：es6
  • 组件的设计与使用：component

# Build Setup

# install dependencies
npm install

# serve with hot reload at localhost:8088
npm run dev

# build for production with minification
npm run build

# build for production with minification
npm run examples
